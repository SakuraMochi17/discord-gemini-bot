# 🤖 Discord Gemini AI Bot

Googleの高性能生成AI「Gemini」をDiscord上で手軽に利用できる、多機能チャットBotです。
単なる一問一答ではなく、チャンネルの文脈（空気を読む）を理解し、画像の読み取り、性格の変更、会話の要約までこなすスマートなアシスタントとして機能します。

## ✨ 主な機能

* **💬 文脈を理解した自然な会話**
  チャンネル内の直近の会話（最大30件）を裏で記憶しているため、「さっきの話だけど〜」といった文脈を踏まえた返答が可能です。
* **🧵 スレッドによる話題の整理**
  Botにメンションをして話しかけると、自動でスレッドを作成して返信します。チャンネルのチャット欄を圧迫せず、個別の話題に集中できます。スレッド内ではメンション不要で会話が続きます。
* **👀 画像認識（マルチモーダル対応）**
  画像を添付してメッセージを送信すると、Geminiがその画像を読み取って内容を解説したり、質問に答えたりします。
* **🎭 キャラクター（性格）変更機能**
  コマンド一つでBotの口調や性格を自由に変更できます。
* **📝 チャンネル会話の要約**
  直近のチャットログを読み取り、「今どんな話題で盛り上がっているか」をサクッと要約して教えてくれます。

---

## 🎮 使い方（チャットの基本）

Botが導入されている指定のチャンネルで、以下の方法で話しかけることができます。

1. **メンションで話しかける (`@Bot 〇〇`)**
   * Botが自動で**専用のスレッドを作成**し、その中で返信します。
   * スレッド内では、メンションなしでそのまま連続して会話が可能です。
2. **キーワードを含めて話しかける (`gemini 〇〇`)**
   * メッセージ内に `gemini` （大文字小文字問わず）が含まれていると、スレッドを作らずにその場で返信します。
   * ※この際、通知音が鳴らない「メンションなしリプライ」でスマートに返信します。
3. **画像を送信する**
   * 画像を添付し、一緒にコメントを添えてBotを呼ぶと、画像の内容について回答します。

## ⌨️ スラッシュコマンド一覧

Discordのチャット欄に `/` を入力すると、以下のコマンドを使用できます。

| コマンド | 説明 |
| :--- | :--- |
| `/summary [limit]` | チャンネルの直近の会話（デフォルト30件、最大100件）を要約して教えます。 |
| `/persona [character]` | Botの性格や口調を変更します。（例：ツンデレ、関西弁、執事など）※記憶はリセットされます |
| `/model [flash/pro]` | 使用するGeminiのモデルを切り替えます。高速な`flash`と、高精度な`pro`を選べます。 |
| `/reset` | あなたとBotとのこれまでの会話履歴（記憶）を消去して、話題をリセットします。 |
| `/stop` | Botのシステムを安全に終了させます。 |

## 🛠️ セットアップ手順（導入方法）

このBotを自身の環境で動かすための手順です。

### 1. 事前準備
以下の3つの情報を取得しておきます。
* **Discord Bot Token**: [Discord Developer Portal](https://discord.com/developers/applications) からBotを作成し、Tokenを取得します。
  * ⚠️ **重要**: Botの設定画面 (Botタブ) で **「Message Content Intent」** を必ずONにしてください。
* **Gemini API Key**: [Google AI Studio](https://aistudio.google.com/) からAPIキーを取得します。
* **Target Channel ID**: Botを動作させたいDiscordチャンネルのID（数字）をコピーします。

### 2. インストール
リポジトリをクローンし、必要なPythonライブラリをインストールします。

```bash
git clone [https://github.com/あなたのユーザー名/あなたのリポジトリ名.git](https://github.com/あなたのユーザー名/あなたのリポジトリ名.git)
cd あなたのリポジトリ名
pip install discord.py google-generativeai pillow python-dotenv
```

### 3.環境変数の設定 (.env)
Botのディレクトリ内に .env という名前のファイルを作成し、事前準備で取得した情報を記述します。
```env
DISCORD_TOKEN=あなたの_Discord_Token
GEMINI_API_KEY=あなたの_Gemini_API_Key
TARGET_CHANNEL_ID=動作させたいチャンネルのID（数字）
```

### 4.起動
以下のコマンドでBotを起動します。
```bash
python bot.py
```

コンソールに Bot名 起動完了！ と表示されれば成功です！

## 📂 ファイル構成
`bot.py`: Botのメインプログラム。

`.env`: トークンなどの秘密情報を保存するファイル（※GitHubには公開しないでください）。

`.gitignore`: Gitの管理から除外するファイルの一覧。

`memory.json`: ユーザーごとの会話履歴（記憶）が自動で保存されるファイルです。起動時に読み込まれ、終了時や会話のたびに更新されます。
